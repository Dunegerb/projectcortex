# Uso responsável

O Projeto Córtex é uma ferramenta educacional de autorregulação. Não é terapia, diagnóstico, tratamento, dispositivo médico nem serviço de emergência.

A linguagem e os cálculos de “recuperação” são metáforas de progresso comportamental, não medições de alterações cerebrais. Um deslize não apaga a trajetória; o produto evita vergonha, punição e promessas de sucesso garantido.

Procure um profissional qualificado quando o comportamento causar sofrimento intenso, prejuízo relevante, compulsão persistente, violência, risco, depressão importante ou pensamentos de autoagressão. Em emergência, use o serviço de emergência local.
